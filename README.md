# Influencer Marketing App

A React Native mobile application that simulates a simple influencer marketing platform.  
Users can browse influencers, view profile details, and explore recent campaign posts.

## Features

- Browse influencers in a clean card layout
- Search influencers by name or location
- Favorite influencers using a star icon
- View influencer profile details
- View recent campaign posts
- Modern UI with custom color palette

## Tech Stack

- React Native (Expo)
- React Navigation
- JavaScript (ES6)
- REST APIs

## APIs Used

Random User API  
https://randomuser.me

JSONPlaceholder API  
https://jsonplaceholder.typicode.com/posts

## App Screens

### Influencer List
Displays influencers with:
- Profile image
- Name
- Location
- Followers
- Engagement rate

### Influencer Profile
Shows:
- Full profile details
- Email
- Followers and engagement
- Recent campaign posts
