# Influencer Marketing App

A **React Native (Expo)** mobile application that simulates a simple **influencer marketing platform**.  
Users can browse influencers, search by name or location, mark favorites, and view influencer campaign posts.

This project demonstrates **React Native fundamentals, API integration, navigation, and clean UI design**.

---

## Features

- Browse influencers in a modern card layout  
- Search influencers by **name or location**  
- Mark influencers as **favorites** using a star icon  
- View **influencer profile details**  
- View **recent campaign posts**  
- Clean and modern UI using a custom color palette  

---

## Tech Stack

- React Native (Expo)
- React Navigation
- JavaScript (ES6)
- REST APIs

---

## APIs Used

**Random User API**  
https://randomuser.me

Used for generating influencer profile data.

**JSONPlaceholder API**  
https://jsonplaceholder.typicode.com/posts

Used for generating sample campaign posts.

## App Screens

### Influencer List Screen

Displays a list of influencers with:

- Profile image
- Influencer name
- Location
- Followers count
- Engagement rate
- Favorite star button

---

### Influencer Detail Screen

Displays detailed information including:

- Profile image
- Full name
- Email
- Location
- Followers count
- Engagement rate
- Recent campaign posts
